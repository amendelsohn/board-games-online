# board-games-online — Design Context

## Design Context

### Users

Casual social players dropping in with friends to play quick rounds of board games in the browser. No signup, share a 4-letter code, play. Expected contexts:

- Friday/weekend evenings at home, on a laptop or TV-cast desktop
- Phone in the kitchen while someone else hosts on a laptop
- Work breaks, someone starts a game of Codenames in Slack
- Kids + parents mixed audience — needs to be friendly, not sterile

The job-to-be-done is *"let me and my friends start playing in the next 15 seconds with zero friction and feel like we chose a premium product"*. The app is an instrument for a social moment, not the moment itself — it should disappear once the game starts, then reappear with delight when the game ends.

### Brand Personality

Three words: **warm, confident, tactile**.

- *Warm* because games are social and shared. No cold, corporate SaaS defaults.
- *Confident* because this should feel curated — like someone cared about every edge.
- *Tactile* because board games are physical objects. The digital version should reference felt, cardstock, hardwood, ivory, lacquer. Pieces should feel like they have weight.

### Aesthetic Direction

**Codename: Parlor.** A well-appointed game room on a weekday evening. Warm amber light, green-felted tables, hand-lettered signage, cardstock with letterpress impressions. Editorial typographic rhythm — like a small-press magazine rather than a product landing page.

Anti-references (what this must NOT look like):
- **netgames.io** — too plain, looks like a side project
- **boardgamearena.com** — cluttered Bootstrap-era UI, visually unmanaged
- Typical "AI-generated SaaS": cyan-on-black gradients, glassmorphism cards, purple-to-blue gradient headings, same 6-card grid with rounded-icon-+-heading, hero-metric layouts
- Dashboard generic: side-stripe alert cards, sparklines-as-decoration

Visual specifics:
- **Type pairing**: *Young Serif* (display, warm slab-serif character) + *Hanken Grotesk* (body, a grotesque with warmth — specifically not Inter/DM Sans) + *JetBrains Mono* (join codes, numeric readouts). This combination is deliberate — none of the three is in the "AI reflex" font pattern.
- **Palette**: Amber primary, plum secondary. Neutrals tinted warm toward amber hue. In light mode: ivory/parchment surfaces, warm graphite text. In dark mode: deep teal-charcoal, warm cream text, golden amber primary that glows like a reading lamp.
- **Spacing**: Editorial rhythm — varied and intentional, not uniform padding. Game boards remain symmetric (precision matters for play); chrome/marketing is asymmetric left-aligned.
- **Motion**: Subtle stagger on page entrance. Satisfying drop/place easing (`cubic-bezier(0.22, 1, 0.36, 1)`). Never bouncy.
- **Detail signature**: Tiny letterpress-style indent-shadow on pill labels and the join code tiles. Subtle paper grain on large surfaces in dark mode. Decorative bullets for section headers. Numeric stats in mono to feel like scorecards.

Both themes are fully supported. Dark defaults via `prefers-color-scheme` since game nights are evening events.

### Design Principles

1. **Game-first, app-second.** In-match, the UI recedes — chrome is compressed, colors are quiet, nothing competes with the board. Out-of-match, the UI takes space and feels editorial.
2. **Pieces feel physical.** Every interactive token (cards, tiles, codes, pieces) has weight — via tone layering, not drop shadows. Motion respects physics.
3. **Typography carries hierarchy, not boxes.** No card-in-card nesting. A heading size change and whitespace do the work of a border.
4. **Warm over clean.** Given a choice between a sterile precision and a warmer decision, pick warm. This distinguishes from every other board-game site.
5. **Mobile is primary.** Everything must work beautifully on a 375px phone held portrait, including all game boards. Adapt layouts, don't amputate features.
6. **Both themes, equally considered.** Dark mode is not a tint of light mode — it has its own palette identity and is treated as a first-class citizen.
